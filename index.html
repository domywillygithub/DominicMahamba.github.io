Honda car sales analysis in Python

Within this project, I have undertaken an in-depth exploratory data analysis of Honda sales data spanning diverse states across the United States. To execute this analysis, I harnessed the capabilities of Google Colab, an open-source platform. This endeavor took place within Google's cloud servers, which allows for the utilization of Google's potent hardware resources such as GPUs and TPUs, irrespective of the computational capacity of your local machine. The beauty of this undertaking lies in its accessibility , a mere browser is all that's required. The truly remarkable facet of this project is its reproducibility; individuals can effortlessly execute the provided code notebook, complete with a link to the dataset, and faithfully replicate the analysis in a seamless manner.
